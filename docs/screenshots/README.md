# Capturas de Tela para os Manuais

Este diretorio guarda as imagens usadas nos manuais Word dos arquivistas.

Nomes esperados pelos geradores:

- `login.png`
- `portal.png`
- `dashboard.png`
- `api_docs.png`

Se os arquivos nao existirem, os scripts criam placeholders automaticamente.

Scripts geradores:

- `docs/doc-gen/gerar_manual_arquivistas_docx.py`
- `docs/doc-gen/gerar_manual_executivo_docx.py`
