
# US-012 — Metadados de Controle e Segurança
**Como** Segurança da Informação  
**Quero** definir perfil de metadados (sigilo, confidencialidade, restrições)  
**Para** controlar o acesso e a proteção dos documentos

## Observações
- Quando houver interface, aplicar identidade visual HW1 (tokens de tema e componentes padronizados).

## Critérios de Aceite (Gherkin)
Cenário: Metadados mínimos por classe
Dado uma classe documental
Quando defino metadados obrigatórios
Então devo incluir proveniência, data de produção, produtor, suporte, nível de sigilo, base legal e retenção aplicada
